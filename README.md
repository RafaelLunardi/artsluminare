# ArtsLuminare

Site estatico da ArtsLuminare, pensado para divulgar e vender amigurumis com visual clean, acessivel e pronto para GitHub Pages.

## Estrutura

- `index.html`: conteudo principal do site
- `styles.css`: estilos, paleta e responsividade
- `script.js`: filtro simples dos produtos
- `assets/`: imagens usadas na vitrine

## Publicacao

O projeto foi preparado para funcionar como site estatico, sem etapa de build.
